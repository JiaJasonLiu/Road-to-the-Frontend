- Section: _Using arbitrary values_
- Applies to layout utilities like `grid-template-columns`
    

In the docs, this syntax is explained generically as:
`utility-[custom-css-value]`

So:
`grid-cols-[auto_1fr]`

maps directly to:
`grid-template-columns: auto 1fr;`

The `[]` are the customer-css-value
